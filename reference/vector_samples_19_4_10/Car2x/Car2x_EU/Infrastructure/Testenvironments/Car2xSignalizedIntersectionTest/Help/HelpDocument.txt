CANoe Car2x Infrastructure Testing
--------------------------------------------------------------------------
With this sample configuration a test can be executed to test an RSU 
sending MAP and SPAT messages. 
For live testing, start the configuration and input the IntersectionID 
of the intersection that should be tested. Start the TestUnit, the results 
can be seen in the TestReport.
For offline testing, activate the Replay-Block and input the logging file.
Configure the IntersectionID of the intersection that should be tested.
