CANoe Car2x Infrastructure Testing
--------------------------------------------------------------------------
With this sample configuration a test can be executed to test a Roadworks
Trailer sending DENM messages.
For live testing, start the configuration and start the TestUnit. Follow
the instructions given in Pop-Up Windows. The results can be seen in the
TestReport.
For offline testing, activate the Replay-Block and input the logging file.
