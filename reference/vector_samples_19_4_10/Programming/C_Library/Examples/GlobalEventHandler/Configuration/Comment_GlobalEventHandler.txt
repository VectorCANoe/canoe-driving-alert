C Library - Global Event Handler
----------------------------------------------------------------------------------------------------
This is an example for a CANalyzer/CANoe C Library. It demonstrates the usage of the general event 
handlers (Loading and unloading of DLL; PreStart, Start and Stop of a measurement)

A detailed description can be found in the online help (see section "Sample Configurations").
