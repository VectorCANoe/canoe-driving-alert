# UX Lighting Module

This directory contains UX lighting control implementations with ADAS integration.

## Components

### ADAS Integration
- ADAS event detection and processing
- Lighting response coordination
- Safety-critical event handling

### Parking Lighting
- Reverse parking detection
- Automated lighting control for parking scenarios
- Ambient lighting patterns

## Features

- Dynamic lighting based on vehicle state
- ADAS event-triggered illumination
- Customizable lighting patterns
- Energy-efficient control algorithms

## Development Guidelines

1. Ensure lighting changes are smooth and non-distracting
2. Prioritize safety-critical lighting events
3. Test under various driving scenarios
4. Validate timing and synchronization

## Testing

Run simulation tests using the configurations in `simulation/test-cases/UX-Lighting/`
