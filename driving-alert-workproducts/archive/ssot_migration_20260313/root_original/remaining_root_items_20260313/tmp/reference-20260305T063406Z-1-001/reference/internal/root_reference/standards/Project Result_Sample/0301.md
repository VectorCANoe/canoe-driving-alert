노드	기능 상세	비고

        Powertrain
Transmission Ctrl	차량 운행 정보를 P(Parking), R(Reverse), N(Neutral), D(Drive) 를 통한 기어 변속 및 signal을 Cluster에 전송
Engine Ctrl	차량의 속도(km/h), 엔진의 온도(ºC) 측정 및 차량의 출력 조절. signal을 Cluster에 전송


        Chassis
Brake Ctrl	페달 각도(0 ~ 60)를 받아 signal 전송
Handle Ctrl	졸음 기능 1단계 시 핸들의 진동 조작
Acceleration Ctrl	페달 각도(0 ~ 60)를 받아 signal 전송


        Body
Emergency LIghts Ctrl	사용자의 입력 및 졸음 기능 1단계 시 데이터 받아 작동. 비상등(On/Off) signal을 Cluter에 전송
Window Ctrl	사용자의 입력 및 졸음 기능 2단계 시 데이터 받아 작동.
CO2 Meter Ctrl	사용자의 입력 및 졸음 기능 2단계 시 데이터 받아 작동 및 CO2 signal을 Cluter에 전송
DSW Ctrl	운전자의 눈을 감지해 졸음 기능 단계를 Body Domain에 있는 Ctrl에 전달


        Infotainment
Cluster Ctrl	Transmission 정보, Engine 상태 정보, 액셀 정보, 비상등 정보, CO2 정보, 카메라 메시지 수신 및 Panel UI 제어


        Actual Device
Emergency Lights	실제 비상등. 비상등(On/Off) signal을 받아서 작동
Cluster	실제 Cluster. UI를 통해 사용자에게 정보 제공
Transmission	Transmission Button. 사용자의 입력을 받는 Button 사용
Acceleration Pedal	페달 각도를 받아 Acceleration Ctrl에 signal 전달
Brake Pedal	페달 각도를 받아 Brake Ctrl에 signal 전달
Window	창문 상태에 따라 창문 개페 상태 조절
