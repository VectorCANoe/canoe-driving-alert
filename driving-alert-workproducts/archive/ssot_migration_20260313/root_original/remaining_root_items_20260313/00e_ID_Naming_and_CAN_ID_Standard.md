# 00e/00f Index (분리 안내)

기존 통합 문서(`명명 + ID`)는 아래 3개 표준으로 분리되었다.

1. ECU 명명/약어 표준: `00e_ECU_Naming_Standard.md`
2. CAN ID 배정 표준: `00f_CAN_ID_Allocation_Standard.md`
3. RTE Name Mapping 표준: `00g_RTE_Name_Mapping_Standard.md`

적용 규칙:

- `03`/개발팀 명칭 변경 지시는 `00e`를 참조한다.
- `0303`/`0302`/ID 블록·충돌·예약 정책은 `00f`를 참조한다.
- `04`/RTE 생성명 규칙·길이 예산·샘플 검토는 `00g`를 참조한다.
- 기존 경로를 참조하는 임시 문서는 순차적으로 갱신한다.
