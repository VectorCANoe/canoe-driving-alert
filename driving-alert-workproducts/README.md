## Excluded From Official Repo Surface

The following paths are working history or local/internal support material and should be excluded from the official/public repo surface:

- `archive/`
- `reference/`
- `ops/`

This exclusion also covers:

- archived `tmp/` material
- local reference bundles
- handoff notes
- mentoring notes
- reports
- change-order history
- other internal notes used during execution and review

## Ownership Rule

- official reviewer-facing workproducts stay in the top-level `00~07` set, `governance/`, and `excel/`
- archive, tmp-like history, local references, and internal execution notes do not belong to the public surface

## Not Owned Here

- active CANoe runtime implementation
- local AGENT working material
- local reference vault for development-only lookup
