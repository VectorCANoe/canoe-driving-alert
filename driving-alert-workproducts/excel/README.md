# Excel Submission Index

이 폴더는 제출용 엑셀 산출물을 보관한다.

## Usage

- 최종 제출에 포함되는 엑셀 파일만 유지한다.
- 원본 작업용 중간 파일은 이 폴더에 두지 않는다.

## Working Rule

- 파일명은 제출 시점 기준으로 고정한다.
- 상세 문서 구조 기준은 상위 `ops/submission/README.md`와 루트 SSoT(`driving-alert-workproducts/`) 실제 파일 목록을 따른다.
- 재생성 스크립트: `excel/build_submission_workbook.py`
