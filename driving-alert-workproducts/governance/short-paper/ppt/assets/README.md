## 원칙

- 모든 자산은 흑백/회색조 기준으로 작성한다.
- PPT에서 깨지지 않도록 과도한 장식 요소를 두지 않는다.
- 각 자산은 한 슬라이드에서 1개만 사용해도 메시지가 전달되도록 만든다.
- 정량 결과 자산은 실측값 입력 전제 템플릿으로 유지한다.
- 데모 발표가 필요할 경우 `_demo.svg` 자산만 사용하고, 실제 측정값과 혼용하지 않는다.
