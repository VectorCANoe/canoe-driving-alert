## Structure

- `paper/`
  - `short_paper_draft.md`
  - `short_paper_working.md`
  - `short_paper_manuscript.tex`
  - `short_paper_twocolumn.tex`
- `poster/`

cd driving-alert-workproducts/governance/short-paper/paper
tectonic short_paper_twocolumn.tex
