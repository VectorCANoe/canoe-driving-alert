# 시스템 테스트 (System Test)

> SDV 기반 차량 경험(Experience) 플랫폼 — E2E 시나리오 테스트

> 실행 원칙: Base Setup(Scene.B) 전체 Pass 후 Test Suite 시나리오(Scene.Z / Scene.O)를 실행한다.

---

## Part 1. Base Setup — 경고 시스템 (Scene.B)

> 운전자의 행동과 차량 반응을 중심으로 서술한다. Scene.B1~B12 전체 Pass = 기본 차량 경고 시스템 완성.

| Scene. ID | 대응 Req | 설명 | Pass/Fail | 담당자 |
|-----------|---------|------|----------|--------|
| **Scene.B1** | Req_B01~B03 | Power OFF 상태로 시작한다. 계기판이 꺼져 있고, 경고등·앰비언트·경고음이 모두 없는 것을 확인한다. | | |
| **Scene.B2** | Req_B01~B03 | Power ON으로 전환한다. 계기판이 켜지며 속도 0 / 경고 없음 / 기어 P 상태가 표시되는 것을 확인한다. 앰비언트는 꺼진 상태이다. | | |
| **Scene.B3** | Req_B01, B04, B09 | 기어를 D로 변경하고 60 km/h까지 가속한다. 제한속도(80 km/h) 이내이므로 경고등·앰비언트·경고음이 없는 것을 확인한다. | | |
| **Scene.B4** | Req_B04, B09~B13 | 속도를 90 km/h까지 올린다. 제한속도(80 km/h) 초과 즉시 계기판 황색 경고등 + "띠딩!" 단발음 + IVI "주의" 메시지가 나타나는 것을 확인한다. 2초 후 운전자 조작 없이 경고가 자동 소거되는 것을 확인한다. | | |
| **Scene.B5** | Req_B05, B09~B13 | 60 km/h 주행 중 급가속 상황을 만든다. 계기판 황색 경고등 + "띠딩!" 단발음이 나타나고 2초 후 자동 소거되는 것을 확인한다. | | |
| **Scene.B6** | Req_B06, B09~B13 | 고속 주행 중 급제동 상황을 만든다. 계기판 황색 경고등 + "띠딩!" 단발음이 나타나고 2초 후 자동 소거되는 것을 확인한다. | | |
| **Scene.B7** | Req_B07, B09~B13 | 정상 주행 중 차선이탈 상황이 발생한다. 계기판 황색 경고등 + "띠딩!" 단발음이 나타나고 2초 후 자동 소거되는 것을 확인한다. | | |
| **Scene.B8** | Req_B08, B09~B13 | 정상 주행 중 급차선변경 상황이 발생한다. 계기판 황색 경고등 + "띠딩!" 단발음이 나타나고 2초 후 자동 소거되는 것을 확인한다. | | |
| **Scene.B9** | Req_B04~B09, B10~B13 | 90 km/h 주행 중 차선이탈이 동시에 발생한다. 경고등이 황색 → 적색으로 바뀌고, 경고음이 연속음(500ms 간격)으로 바뀌며, 앰비언트가 주황 파동 패턴으로 켜지는 것을 확인한다. 두 조건 중 하나가 해소되면 경고가 자동 소거되는 것을 확인한다. | | |

| **Scene.B11** | Req_B15 | **3단계 경고(Scene.B10) 상태에서** 운전자가 전방을 바라본다. 모든 경고등이 꺼지고, 경고음이 멈추며, 앰비언트가 꺼지는 것을 확인한다. | | |
| **Scene.B12** | Req_B16 | **3단계 경고(Scene.B10) 상태에서** 운전자가 핸들을 조작한다. 모든 경고등이 꺼지고, 경고음이 멈추며, 앰비언트가 꺼지는 것을 확인한다. | | |

## Part 2. Test Suite 1 — 준영 (구간 인식 + Ambient 연동, Scene.Z)

> 전제조건: Scene.B1~B12 전체 Pass.

| Scene. ID | 대응 Req | 설명 | Pass/Fail | 담당자 |
|-----------|---------|------|----------|--------|
| **Scene.Z1** | Req_Z01, Z02 | 스쿨존 구간에 진입한다. 속도를 35 km/h로 주행한다(스쿨존 제한 30 km/h 초과). 계기판 황색 경고등이 켜지는 동시에 앰비언트가 빨간색으로 빠르게 점멸하며 과속 경고를 표시하는 것을 확인한다. 속도를 25 km/h로 줄이면 앰비언트 경고가 해제되는 것을 확인한다. | | |
| **Scene.Z2** | Req_Z01, Z03 | 고속도로 구간에 진입하여 110 km/h로 주행한다. 핸들 조작 없이 직진 10초 유지 후 앰비언트가 주황색 파동 패턴으로 바뀌고 시트에 진동이 발생하는 것을 확인한다. 핸들을 조작하면 경고가 즉시 해제되는 것을 확인한다. | | |
| **Scene.Z3** | Req_Z01, Z04 | 고속도로 IC 출구 근접 구간에 진입한다. Panel에서 진출 방향(좌/우)을 설정한다. 앰비언트가 IC 유도선 색상(초록/분홍/빨강)으로 전환되고 설정 방향으로 흐르는 것을 확인한다. 구간을 벗어나면 앰비언트가 즉시 꺼지는 것을 확인한다. | | |
| **Scene.Z4** | Req_Z01~Z04 | 일반도로 → 스쿨존 → 고속도로 → IC출구 → 일반도로 순서로 구간을 연속 전환한다. 각 구간 진입 시 앰비언트 패턴이 즉시 해당 구간 패턴으로 전환되고, 이전 구간 패턴이 완전히 사라지는 것을 확인한다. | | |

## Part 3. Test Suite 2 — 성현 (SOTA OTA 구독 서비스, Scene.O)

> 전제조건: Scene.B1~B12 전체 Pass.

| Scene. ID | 대응 Req | 설명 | Pass/Fail | 담당자 |
|-----------|---------|------|----------|--------|
| **Scene.O1** | Req_O01 | 차량을 P 기어 상태로 정차한다. IVI 화면에서 구독 메뉴를 열고 [Drive Coach 설치] 버튼을 선택한다. 동의 화면이 표시되고, 동의하면 OTA_Server가 ETH_OTA_Param(Port 6000, 8 bytes)을 즉시 전송하는 것을 확인한다. OTA 진행 중 P 기어 이탈 시 세션이 즉시 중단되는 것을 확인한다. | | |
| **Scene.O2** | Req_O01 | Drive Coach 파라미터가 정상 적용된다. IVI에 "Drive Coach 적용 완료" 팝업이 표시된다. 이후 급차선변경 시 LDW 경고 강화 점등 / 급가속 시 토크 제한 적용(반응 제한) / 100 km/h 초과 불가 / 후진 10 km/h 초과 불가를 확인한다. | | |
| **Scene.O3** | Req_O02 | Drive Coach 적용 후 Smart Claim이 활성화된 상태에서 gCrashEvent = 1을 발생시킨다. Python CANoe COM API가 충돌 데이터(타임스탬프/속도/경고 이력)를 Flask 서버로 전송하는 것을 확인한다. IVI에 "사고 데이터 전송 완료" 메시지가 표시되는 것을 확인한다. | | |
| **Scene.O4** | Req_O03 | OTA 서버에서 Seasonal Theme 패키지가 가용함을 알린다. IVI에 알림이 표시되고, 운전자 동의 후 OTA_Server가 ETH_OTA_Param(PackageID=0x02, ThemeID 포함)을 전송한다. 파라미터 적용 즉시 앰비언트 색상 프리셋과 IVI 배경이 변경되는 것을 확인한다. | | |
| **Scene.O5** | Req_O05 | OTA 파라미터 전송 중 CRC8 오류를 주입한다. 파라미터가 적용되지 않고 이전 상태가 유지된다. IVI에 "OTA 실패 — 이전 설정 유지" 메시지가 표시되는 것을 확인한다. | | |

## 시나리오 구조 요약

```
[Base Setup — Scene.B1~B12]
  B1. Power OFF → B2. Power ON 초기 상태 확인
  B3. 정상 주행 (60km/h, 경고 없음)
  B4. 과속 → Level 1 경고 → 2초 자동 소거
  B5. 급가속 → Level 1 경고 → 2초 자동 소거
  B6. 급제동 → Level 1 경고 → 2초 자동 소거
  B7. 차선이탈 → Level 1 경고 → 2초 자동 소거
  B8. 급차선변경 → Level 1 경고 → 2초 자동 소거
  B9. 과속 + 차선이탈 동시 → Level 2 경고 → 조건 해제 시 자동 소거
  B10. gCrashEvent = 1 → Level 3 경고 → 자동 소거 없음
  B11. [3단계] 운전자 전방 응시 → 경고 해제
  B12. [3단계] 운전자 핸들 조작 → 경고 해제
       │
       ├─ [Test Suite 1 — Scene.Z1~Z4] gRoadZone 패널 버튼 추가
       │    Z1. 스쿨존 → 과속 시 Ambient RED 점멸
       │    Z2. 고속도로 → 핸들 미조작 10초 → ORANGE 파동 + 진동
       │    Z3. IC출구 → Ambient 방향 안내 흐름
       │    Z4. 구간 연속 전환 → 패턴 즉시 전환 확인
       │
       └─ [Test Suite 2 — Scene.O1~O5] OTA 구독 버튼 추가
            O1. P 기어 → Drive Coach 버튼 → OTA_Param 전송 확인
            O2. Drive Coach 파라미터 즉시 적용 확인
            O3. Smart Claim → 충돌 데이터 Flask 전송 확인
            O4. Seasonal Theme → 테마 즉시 적용 확인
            O5. CRC8 오류 → 이전 설정 유지 확인
```
