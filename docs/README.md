# Internal Project Records

This folder is not the public documentation surface.

It contains internal project records such as:

- meeting notes
- mentoring notes
- planning and blog-style working material

## Subfolders

- `meeting-notes/`
  - dated meeting records
- `mentoring/`
  - mentoring analysis and guidance notes
- `planning_and_Blog/`
  - planning drafts and blog-style working notes

## Use This Folder When

- you need historical meeting context
- you need mentoring rationale behind an existing decision
- you are tracing how a planning direction changed over time

If you want the active project surface instead, go back to:

- `README.md`
- `product/sdv_operator/README.md`
- `canoe/README.md`
- `driving-situation-alert/README.md`
