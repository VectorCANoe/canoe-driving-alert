# Reference Priority Note

Use `canoe/reference/` as the main local reference bank.

Recommended first picks:

1. `standards/` for policy and reviewer wording
2. `internal-root/` for sample sheets and DBC examples
3. `opensource/cantools` for DBC parsing and generator ideas
4. `opensource/canmatrix` for matrix conversion and mapping ideas
5. `industry/` for quick concept refresh

Do not treat `opensource/` or `industry/` as project source of truth.

Project source of truth remains:

- `canoe/src/capl/**`
- `canoe/project/sysvars/project.sysvars`
- `canoe/docs/**`
- active `canoe/databases/*.dbc`

- `canoe/reference/index/REFERENCE_PRIORITY_GUIDE.md`
