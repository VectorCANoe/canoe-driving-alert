# Retired Test Unit Skeletons

This folder contains draft skeleton assets that are no longer part of the current active naming and build baseline.

## Current rule

1. Keep only active anchor assets and current official-scope skeletons at `test_units/` root.
2. Move superseded draft skeletons into `retire/`.
3. Do not delete retired assets unless the team explicitly decides they are no longer needed as reference.

## Retired scope

The retired assets in this folder remain useful as:

- reference examples
- historical draft intent
- temporary source material for future migration

They are not part of the current official build order.
