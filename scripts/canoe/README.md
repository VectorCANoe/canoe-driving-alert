# CANoe Helper Scripts

Low-level CANoe maintenance helpers:

- `attach_capl.py`
- `check_nodes.py`
- `fix_nodes.py`
- `reload_cfg.py`
- `reload_wait.py`
- `setup_canoe_config.py`

These are maintenance utilities, not daily verification commands.

For normal verification flow, use:

- `python scripts/run.py verify-prepare ...`
- `python scripts/run.py verify-smoke ...`
- `python scripts/run.py verify-fill-score ...`
