## 시나리오 상세 설명

### Scene. 5: 속도 기반 조명 색상 변경

**사전 조건**: 시동 ON, 기어 D, 스포츠 모드 선택

**테스트 순서**:
1. 차량 속도 0km/h에서 시작
2. 속도 30km/h까지 증가 → 앰비언트 조명 녹색 확인
3. 속도 70km/h까지 증가 → 앰비언트 조명 파란색 확인
4. 속도 120km/h까지 증가 → 앰비언트 조명 빨간색 확인

**예상 결과**: 속도 구간에 따라 조명 색상이 부드럽게 전환 (응답시간 < 500ms)

---

### Scene. 9: 후진 안전경고 UI 및 시트조명

**사전 조건**: 시동 ON, 기어 D, 주행 중

**테스트 순서**:
1. 기어 D → R 변경
2. 클러스터 경고 UI 표시 확인 (응답시간 < 300ms)
3. 시트조명 자동 점등 확인
4. 3초 이상 유지 확인
5. 조명 제어 실패 시 음성 경고 출력 확인

**예상 결과**: 후진 진입 즉시 경고 UI 및 시트조명 활성화, 최소 3초 유지

---

### Scene. 16: 긴급 제동 (AEB) 고위험 경고

**사전 조건**: 시동 ON, 기어 D, 속도 60km/h 주행 중

**테스트 순서**:
1. 전방 장애물 급접근 시뮬레이션
2. ADAS Control ECU에서 AEB 이벤트 생성 (AEB_Event=1, AEB_Level=3)
3. vECU에서 이벤트 수신 (응답시간 < 50ms)
4. 대시보드 고위험 경고 UI 즉시 표시 확인 (빨간색, 큰 아이콘)
5. 긴급 제동 종료 시 경고 UI 자동 종료 확인

**예상 결과**: AEB 이벤트 발생 후 50ms 이내 고위험 경고 UI 표시

---

### Scene. 23: Fail-Safe 모드 진입

**사전 조건**: 시동 ON, 조명 제어 정상 동작 중

**테스트 순서**:
1. Lighting Control ECU로 제어 명령 전송
2. 제어 실패 응답 시뮬레이션 (1차)
3. 재시도 제어 명령 전송
4. 제어 실패 응답 시뮬레이션 (2차)
5. Fail-Safe Manager에서 오류 감지
6. Fail-Safe 모드 진입 확인 (응답시간 < 1초)
7. 모든 조명 50% 밝기로 설정 확인
8. FailSafeMode 시스템 변수 = 1 확인

**예상 결과**: 2회 연속 제어 실패 후 1초 이내 Fail-Safe 모드 진입, 조명 50% 밝기 유지

---

### Scene. 28: OTA 업데이트 후 기능 검증

**사전 조건**: OTA 업데이트 완료

**테스트 순서**:
1. 시동 ON
2. 조명 제어 기능 테스트 (색상 변경, 밝기 조정)
3. ADAS 연계 경고 기능 테스트 (LDW, AEB)
4. 진단 기능 테스트 (UDS 서비스)
5. Fail-Safe 기능 테스트
6. 모든 기능 정상 동작 확인

**예상 결과**: OTA 업데이트 후 모든 요구사항 기능이 사전 정의된 대로 정상 동작

---

### Scene. 35: 전체 기능 연계 종합 테스트

**사전 조건**: 시스템 초기화 완료

**테스트 순서**:
1. 시동 ON
2. 프로필 선택 (프로필 2)
3. 모드 선택 (스포츠)
4. 주행 시작 (기어 D)
5. 속도 증가 (0 → 120km/h) → 조명 색상 변경 확인
6. 기어 변경 (D → R) → 후진 경고 확인
7. 후방 장애물 감지 → 조명 점멸 확인
8. 기어 변경 (R → D) → 경고 종료 확인
9. 차선 이탈 발생 → LDW 경고 확인
10. 긴급 제동 발생 → AEB 경고 확인
11. 온도 변경 (18°C) → 조명 색상 변경 확인
12. 조명 제어 실패 시뮬레이션 → Fail-Safe 모드 진입 확인
13. OTA 업데이트 수행 → 업데이트 완료 확인
14. 시동 OFF

**예상 결과**: 모든 시나리오가 순차적으로 정상 동작, 에러 없이 전체 테스트 완료
