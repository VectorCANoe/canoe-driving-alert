# 콘셉트 설계서 (Compact Version)

> 이 문서는 `Project_Result_Sample.xlsx` 형식에 맞추어 현업 실무진 및 아키텍처 전문가 보고용으로 극히 간략화된 버전입니다.

| 구분 | 내용 |
|---|---|
| 프로젝트명 | SDV 기반 차량 경험(Experience) 플랫폼 (3종 HMI/FoD 세트) |
| 핵심 컨셉 | 구간 환경 연동 앰비언트, 초보자 맞춤 주행 보조 구독(Drive Coach), 시즌 테마 개인화(OTA) |
| 타겟 고객 | 안전 보조장치가 필요한 초보 운전자 및 차량 내부 개인화를 원하는 운전자 |
| 주요 기능 | - 일반/스쿨/고속/IC 구간 별 맞춤 반응 HMI<br>- IVI 스토어를 통한 사용자 주도형 Drive Coach 다운로드 (안전 제어 파라미터 업데이트)<br>- Drive Coach 연동 Smart Claim 텔레매틱스 (사고 시 EDR 데이터 보험사 전송)<br>- 시즌별 앰비언트/사운드/클러스터 테마 OTA 업데이트 |
| 비즈니스 목표 | 기능안전(ISO 26262)을 위반하지 않는 선에서 차량의 즉각적인 상품성(UX)을 향상시키는 FoD(Feature on Demand) 수익 구조 실증 |
